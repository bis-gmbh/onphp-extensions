## Onphp Extensions Package

### Installation

In `composer.json` add:

```
    "repositories": [
        ...
        {
            "type": "vcs",
            "url": "git@github.com:bis-gmbh/onphp-extensions.git"
        },
        ...
    ],
    "require": {
        ...
        "bis-gmbh/onphp-extensions": "dev-master",
        ...
    },
```
 and run `composer install`.
